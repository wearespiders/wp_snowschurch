This is a small plugin to customize your icon menus.

== Description ==

This is a small plugin to customize your icon menus. You can change menu icon very easy just click, choose and set to icon.

== Installation ==

1. Activate the plugin
2. Go to Appearance > Menus
3. Click add an icon Or remove an icon and Click save menu
4. Enter to wp-content/plugins/sanooksookjai-menu-icon/customize.css Or themes/yourtheme/style.css and customization about menu items, Example: .menu li a img{ width: 20px; height: auto; } (We included class and id to img, you can click to inspect them.)
5. Enjoy!
